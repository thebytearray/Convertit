**ConvertIt** is an **ad-free** and **easy-to-use** audio converter app that helps users convert audio files across different formats seamlessly. With **support for popular audio formats** and **customizable bitrates**, ConvertIt delivers high-quality conversions, powered by FFmpeg, all within a clean, user-friendly interface.

---

## üöÄ Features

- üéß **Audio Format Conversion**: Effortlessly convert between **FLAC, ALAC, MP3, WAV, AAC, OGG, M4A, AIFF, OPUS, WMA, MKA, and SPX**.
- üéöÔ∏è **Bitrate Selection**: Choose from bitrates such as **64k, 96k, 128k, 192k, 256k, 320k, 512k, 768k, and 1024k** for output quality.
- üì± **Ad-Free**: Enjoy a completely **ad-free** experience.
- üîß **FFmpeg Integration**: Built-in FFmpeg ensures **reliable and high-quality** conversions.
- üìê **Modern Architecture**: Developed using **Kotlin** with **MVVM architecture** for better scalability and maintainability.
- üß≠ **Smooth Navigation**: Implemented using **Android Navigation Component** for easy and smooth in-app navigation.
- üñ•Ô∏è **User-Friendly UI**: Designed with simplicity and ease of use in mind, offering a clean and intuitive interface.
